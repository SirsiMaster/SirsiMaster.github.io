Security Quick Reference

🚀 Quick Start

1. Add Security Scripts

<!-- Add to your HTML <head> --> <script src="/SirsiNexusPortal/assets/js/secure-auth.js"></script> <script src="/SirsiNexusPortal/assets/js/auth-integration.js"></script>

2. Initialize on Page Load

document.addEventListener('DOMContentLoaded', async () => { await SecureAuth.init(); });

3. Add to Forms

<form data-secure-form="login"> <!-- Your form fields --> </form>

🔧 Console Commands

Authentication

// Check authentication status SecureAuth.isAuthenticated()
// Get current user SecureAuth.getCurrentUser()
// Force logout SecureAuth.logout()

Session Management

// Check session validity SecureAuth.validateSession()
// Extend session SecureAuth.extendSession()
// Get session info SecureAuth.getSessionInfo()

Security Checks

// Check permission SecureAuth.checkPermission('resource', 'action')
// Verify CSRF token SecureAuth.validateCSRFToken()
// Get security status SecureAuth.getSecurityStatus()

Debugging

// Enable debug mode SecureAuth.debug = true
// View audit logs SecureAuth.getAuditLogs()
// Clear all data SecureAuth.clearAllData()

🧪 Testing Procedures

Test Login Flow

  1. Open browser console (F12)
  2. Navigate to login page
  3. Run: SecureAuth.debug = true
  4. Fill and submit login form
  5. Check console for security events
  6. Verify: SecureAuth.isAuthenticated() returns true

Test Session Timeout

// Set short timeout for testing (5 seconds) SecureAuth.init({ sessionTimeout: 5000 }) // Login and wait await SecureAuth.login({ email: 'test@example.com', password: 'test' }) // Check after timeout setTimeout(() => { console.log('Session valid:', SecureAuth.isAuthenticated()) }, 6000)

Test CSRF Protection

// Get current CSRF token const token = SecureAuth.getCSRFToken() console.log('CSRF Token:', token) // Verify form has token const form = document.querySelector('[data-secure-form]') const csrfInput = form.querySelector('input[name="csrf_token"]') console.log('Token in form:', csrfInput?.value === token)

⚠️ Common Issues & Solutions

HTTPS Required Error

For local development only:

SecureAuth.init({ requireHttps: false })

IndexedDB Not Available

Check if running in private mode:

if (!window.indexedDB) { console.warn('IndexedDB not available, using fallback') }

Session Expired Unexpectedly

Listen for expiry events:

window.addEventListener('auth:session:expired', (e) => { console.log('Session expired:', e.detail) // Handle gracefully })

📡 Event Reference

Event Description Detail
auth:init Security service initialized { config }
auth:login:success User logged in successfully { user, session }
auth:login:failed Login attempt failed { error, attempts }
auth:logout User logged out { userId }
auth:session:expired Session timeout reached { sessionId }
auth:security:threat Security threat detected { type, details }
