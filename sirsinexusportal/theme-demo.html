Fresh Theme Toggle Demo

Clean Theme Toggle System

This demo showcases our fresh theme toggle implementation with no external dependencies, clean localStorage persistence, and dynamic sun/moon icons.

Flash Prevention

Inline script in the head prevents flash of unstyled content by setting the theme before page render.

LocalStorage Persistence

Simple localStorage-based theme persistence that remembers user preference across sessions.

Dynamic Icons

Theme toggle button shows sun icon for light mode and moon icon for dark mode automatically.

No Dependencies

Clean implementation with no external dependencies or complex event systems to avoid conflicts.

Usage Instructions

1. Add Theme Toggle Button:

<button data-theme-toggle>Toggle Theme</button>

2. Include Fresh Theme Toggle Script:

<script src="/assets/js/fresh-theme-toggle.js"></script>

3. Add Dark Mode Classes:

class="bg-white dark:bg-slate-900 text-slate-900 dark:text-slate-100"

Test the Theme Toggle

Click the theme toggle button in the header above to switch between light and dark modes. Your preference will be saved and restored on page refresh.
